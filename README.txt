I'm John Winborne and this project is my final assignment for CS 463/563 – Intro to Web Development.
This website is a professional personal portfolio showcasing who I am, my experience, and the projects I have built.

The site has a navigation bar, landing page, About section, Previous Work section, Projects section, Contact section, and an interactive JavaScript feature (a custom light/dark mode toggle using my logo)

Project Overview

This is a single-page personal portfolio built with HTML, CSS, and JavaScript.
It uses:

Good and accessible HTML

Custom CSS styling and responsive layouts

JavaScript interactivity using the DOM

Smooth scrolling navigation

A custom light/dark mode implemented through class toggling

A contact form (non-functional)

Integration of external assets, icons, and my resume
